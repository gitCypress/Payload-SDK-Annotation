## 角色划分
* 负载开发：

  将各类专用设备与无人机的飞行平台相结合，拓展无人机的负载能力。如PSDK云台相机、PSDK气体检测仪等。

* 应用开发：
  
  基于计算机模块开发各类识别与控制类算法应用，提升无人机的智能化水平。如室内飞行应用、桥梁自动化巡检应用。
